# Using pip

## Create a virtual environment


<!-- termynal -->

```
$ mkdir my_project
$ cd my_project
$ python3 -m venv my_venv
```

## Activate the environment and install using pip

<!-- termynal -->

```
$ source my_venv/bin/activate
$ pip install drops-py
---> 100%
Installed
```
